# Unimad Frontend V1

This is a Next.js project and uses inline Tailwind CSS.

Default package manager is Pnpm.

# Node

Node Version is > 18.0.0

# To Start Project

Install the pnpm by `npm install -g pnpm`

```
1. pnpm install
2. pnpm build
3. pnpm start
```

or

```
1. pnpm install
2. pnpm dev
```
